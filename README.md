# chado (茶道)

**The way of tea**

A mindful TUI for [jujutsu](https://github.com/martinvonz/jj). The way of tea emphasizes presence, precision, and practice — so too with version control.

## Installation

```bash
go install github.com/chatter/chado@latest
```

## Usage

```bash
cd your-jj-repo
chado
```

## Keybindings

| Key | Action |
|-----|--------|
| `j` / `k` | Navigate up/down |
| `h` / `l` | Switch panes |
| `Enter` | Drill into files |
| `Esc` | Go back |
| `{` / `}` | Previous/next hunk |
| `g` / `G` | Top/bottom |
| `q` | Quit |
